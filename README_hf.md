# 🎨 Carousel Generator

Générateur de carrousels Instagram professionnels avec import Markdown intelligent.

## ✨ Fonctionnalités

- **Import Markdown** : Glissez-déposez ou collez du contenu Markdown
- **19 thèmes premium** : Palettes de couleurs harmonieuses
- **Mode Présentation** : Plein écran avec navigation clavier
- **Export PNG & PDF** : Téléchargement individuel ou ZIP
- **Texte auto-adaptatif** : Taille optimale selon le contenu

## 🚀 Utilisation

1. **Importez** votre fichier Markdown (.md)
2. **Choisissez** un thème (ou aléatoire)
3. **Générez** le carousel
4. **Téléchargez** en PNG ou PDF

## 📖 Documentation

Voir le fichier `DEPLOYMENT_GUIDE.md` dans le repository pour plus de détails.

## 🛠️ Technologies

- **Backend** : FastAPI + Uvicorn
- **Templates** : Jinja2
- **Rendu** : Playwright (Chromium)
- **Frontend** : HTML/CSS/JS vanilla
